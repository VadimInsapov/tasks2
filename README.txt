Технологии проекта:
Язык прогроммирования PHP
База данных MySQL

домен https://tasks2

Для подключения своей базы данных нужно поменять строку подключения в vendor/connect.php

Результат работы программы содержится в папке screenshots, данные из базы данных также содержаться в данной папке

Все требования придъявленные в задании были выполнены, за некоторыми исключениями:
    1. при регистрации проверяется существование пользователя с вводимым логином, если пользователь с таким логином уже существет то выдается ошибка
    2. разными цветами отображаются не заголовки задач, а их блоки с названием и описанием
    3. при создании задачи реализовано задание начальной даты не раньше нынешнего времени
    4. при клике по блоку с задачей, выходит подробное описание задачи с описанием, там же есть возможность перейти на редактирование этой задачи
   

Валидация форм при регистрации происходит при помощи php, валидация форм при отправлении задачи и обновлении её происходит  при помощи js,jquery

Файлы, с версткой и отображением страниц находятся в основном катологе проекта
В папке vendor содержится весь код который управляет данным
-connect.php -происходит подключение к базе данных
Регистрация, авторизация и выход из авторизованной страницы происходит в данных файлах, сохранение сессии происходит при помощи cookie
-logout.php
-signin.php
-signup.php
Добавление задачи в базу данных
-addTask.php
Редактирование задачи в базе данных
-updateStatus.php
-updateTask.php
Получение задач из базы данных
-filterDate.php 
-filterSub.php
-queries.php

Папка js
-choice-list.js открытие и выбор элемента из выпадающего списка
-main.js устаовление минмальной даты при создании задачи и окрашивание блоков задач под требуемые цвета
-popup.js открытие и закрытие модального окна
-validateForm.js валидация форм
